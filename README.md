# Novel Storytelling Framework Documentation

## 概要
このフレームワークは、成年向けコンテンツを含む小説を創作するためのガイドラインとドキュメントを提供します。キャラクター、ストーリーライン、設定、舞台などの要素を矛盾なく管理し、人を惹きつける物語を作成するためのベストプラクティスをまとめています。

## 目的
- 物語の整合性を保つ
- エロティックな要素を効果的に取り入れる
- 読者を引き込むスタイルを実現する

## 主要コンポーネント

### 1. キャラクター (Character)
キャラクターは物語の中心。以下の属性を定義してください：
- **名前**: 日本語表記（例: アリス）
- **年齢**: 数値
- **性別**: Male/Female など
- **背景**: 簡単な説明
- **外見**: 詳細な描写（髪、目、体型など）
- **性格**: 性格の特徴
- **スキル**: 能力や特技
- **出身地**: 生まれた場所
- **育ち方**: どのように育ったか
- **価値観**: 何を重視するか
- **好きなもの**: 好むもの
- **嫌いなもの**: 嫌うもの
- **関係**: 他のキャラクターとのつながり

### 2. ストーリーライン (Storyline)
物語のプロット。以下の要素を含めてください：
- **タイトル**: 物語の名前
- **プロットポイント**: 主要な出来事のリスト

### 3. 設定 (Setting)
世界観とルール：
- **世界**: ファンタジー、現代など
- **ルール**: 魔法の有無など
- **スタイル**: 書き方のガイド（例: ライトエロティック小説スタイル）

### 4. 舞台 (Stage)
シーンごとの場所と時間：
- **場所**: 森、街など
- **時間**: 中世など

### 5. 矛盾チェック (Consistency Checker)
要素間の整合性を確認：
- 年齢と背景の一致
- 設定とキャラクターの適合

## 書き方のガイドライン

### 視点
- 第一人称視点を使用: 主人公の内面描写を重視し、読者の没入感を高める。
- ユーモアを交える: 軽いトーンで物語を進め、読者を飽きさせない。

### エロティック要素
- **トラップの活用**: エロトラップを導入し、予期せぬ状況で性的描写を展開。
- **詳細な描写**: 感覚（触感、音、匂い）、感情、動きを細かく記述。
- **ビルドアップ**: 緊張を高め、クライマックスへつなげる。
- **キャラクターの反応**: 表情、声、思考を描写してリアリティを出す。

### スタイルの例
サンプル小説（「表情豊かな無口聖女とエロトラップダンジョン攻略」）を参考に：
- ライトノベル風: 対話多め、情景描写。
- エロ描写: 羞恥と快楽のバランス。
- 長さ: 章あたり3000字程度を目指す。

## 使用方法
1. キャラクターを定義: 詳細な属性を設定。
2. ストーリーラインを作成: プロットをリストアップ。
3. 設定と舞台を決める: 世界観を固める。
4. 章を書く: 第一人称でトラップ要素を交え、エロ描写を詳細に。
5. 矛盾チェック: 要素が一致するか確認。

## 注意点
- 成年向けコンテンツ: フィクションとして扱い、適切な表現を使用。
- 整合性: キャラクターの背景と行動が矛盾しないよう注意。
- 読者引き込み: ユーモアとサスペンスを組み合わせる。

## サンプルストーリー
- タイトル: The Forbidden Love
- 主人公: アリス（25歳、冒険者）
- 相手: ボブ（30歳、魔法使い）
- 舞台: Enchanted Forest
- プロット: 出会い → トラップ発動 → 愛の始まり

このドキュメントを基に、創造的な小説創作をお楽しみください。
